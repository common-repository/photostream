Photostream

== Description ==

Plugin to display a list of photo's as a stream. Uses slimbox to display bigger photos'.

== Installation ==

Usage: create a folder in your WP-root named 'albums'. Drag your photo's in there. 
Thumbnails are recognized by adding 'thumb_' as prefix in the filename.
